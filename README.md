## 🛠️ Tech Stack

**Frontend:**
- Next.js 15+, React 19, TypeScript
- Tailwind CSS, ShadCN UI Components
- Chart.js / Recharts for visualizations
- WebSocket for Real-time data

**Backend:**
- Node.js 20+, Next.js API Routes
- Server Actions for data fetching
- Bull Queue for background jobs
- Redis for caching

**Database:**
- Supabase PostgreSQL (Primary)
- Firebase Firestore (Optional)
- Redis for session & cache

**AI/ML:**
- OpenAI API (GPT-4 for reasoning)
- Google Gemini API (Backup)
- TensorFlow.js (Browser ML)
- Custom ML Models (NumPy/Scikit-learn)

**Authentication:**
- Firebase Authentication
- Google OAuth 2.0
- Email/Password auth

**Hosting:**
- Vercel (Frontend + Serverless Backend)
- Firebase Cloud Functions (Optional)
- AWS EC2 (Scaling layer)
- Docker Containers

The platform uses 7 specialized AI agents:

1. **Market Structure Analyst** - Trend, structure, breakout analysis
2. **Options Flow Analyst** - Option chain, OI, PCR, Max Pain analysis
3. **Institutional Activity Analyst** - FII/DII, volume, smart money tracking
4. **Stock Selection Analyst** - Screening, momentum, relative strength
5. **Risk Management Analyst** - Position sizing, stop loss, risk-reward
6. **Market Sentiment Analyst** - VIX, breadth, sector analysis
7. **Master Decision Engine** - Collects insights, generates final signal

## 🚀 Quick Start

See **[IMPLEMENTATION_GUIDE.md](./docs/17_IMPLEMENTATION_GUIDE.md)** for detailed setup instructions.

### Prerequisites
- Node.js 20+
- PostgreSQL database
- Firebase account
- OpenAI API key
- NSE data source subscription

### Basic Setup
```bash
# Clone repository
git clone https://github.com/praveenrai-hue/nse-trading-intelligence-platform.git
cd nse-trading-intelligence-platform

# Install dependencies
npm install

# Setup environment
cp .env.example .env.local

# Run development server
npm run dev

# Visit http://localhost:3000
```
